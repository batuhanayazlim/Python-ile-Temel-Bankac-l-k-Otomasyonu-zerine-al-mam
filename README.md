# Python-ile-Temel-Bankac-l-k-Otomasyonu-zerine-al-mam
Python öğrenme sürecimde, bir bankamatik (ATM) sisteminin arka planda nasıl çalıştığını simüle eden interaktif bir "Banka Yönetim Sistemi" projesi geliştirdim.   Bu projede sadece veri giriş-çıkışı değil, aynı zamanda gerçek hayattaki bankacılık mantığını (Bakiye kontrolü, ek hesap limit yönetimi, kullanıcı doğrulama) koda döktüm.
